# Módulo 25 - Testes Mobile com Appium

Repositório criado para a entrega da atividade prática do Módulo 25 da EBAC, com foco na validação do ambiente para testes mobile utilizando Appium, Android Studio e Appium Inspector.

## Objetivo

Confirmar que o ambiente de testes mobile está configurado corretamente para automação Android.

## Ferramentas verificadas

- Node.js
- NPM
- Java JDK
- Android Studio
- Android SDK
- ADB
- Appium
- Appium Doctor
- Driver UiAutomator2
- Appium Inspector
- Emulador Android

## Evidências

O arquivo principal da entrega está disponível em PDF:

- `Validacao_Ambiente_Richard_Balestrim (3).pdf`

O PDF contém prints de validação do ambiente, incluindo:

- Versões do Node.js, NPM, Java e Javac
- Validação do ADB
- Variáveis de ambiente JAVA_HOME e ANDROID_HOME
- Resultado do Appium Doctor
- Instalação do driver UiAutomator2
- Emulador Android em execução
- Configuração do Appium Inspector
- Projeto Android aberto no Android Studio

## Status da entrega

Ambiente validado com sucesso para os próximos passos de automação mobile com Appium.
